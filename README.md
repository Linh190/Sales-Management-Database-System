# Sales Management Database System – Điện Máy Xanh
## Overview

This project was developed for the Database Management Systems course and focuses on designing and implementing a Sales Management Database System for a retail electronics business environment inspired by Điện Máy Xanh.

The system supports sales operations, inventory management, supplier management, invoice processing, and reporting activities using SQL Server and Power BI.

## Objectives
Design and normalize a relational database system
Manage sales, inventory, suppliers, and invoices efficiently
Implement SQL programming components for automation and security
Optimize query performance for large datasets
Visualize business insights using Power BI dashboards
